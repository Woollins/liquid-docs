With Liquid you can create a bespoke eCommerce solution.

What is Liquid?

Liquid is a templating language. This means that we can access data from our shop (e.g product title, collection description, product images or blog posts) and output that directly to our templates. One of the main benefits is that we don't need to have any knowledge of what that data is, wejust need to know which variables we have access to in each template.

Tags

Liquid tags are used to tell the template what to do. You can control if code gets executed, repeatedly run blocks of code and create newvariables.

Models

Liquid models (or objects) contain attributes to output dynamic content on the page. Liquid models are also often referred to as Liquid variables.

Filters

Liquid filters are simple methods that modify the output of numbers, strings, variables and objects.
